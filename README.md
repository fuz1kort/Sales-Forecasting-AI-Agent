# Sales Forecasting AI Agent 🤖📊

Интеллектуальный агент для анализа и прогнозирования продаж с использованием ансамбля моделей машинного обучения.

## ✨ Возможности

- **📈 Прогнозирование продаж** - Ансамбль из 3 моделей для точных прогнозов
- **🧪 Анализ стационарности** - ADF-тест для проверки временных рядов
- **📊 Визуализация данных** - Корреляции, распределения, ABC-анализ, временные ряды
- **🏆 Сравнение моделей** - Бэктестирование и выбор лучшей модели
- **📦 Анализ товаров** - KPI, тренды, сезонность, топ-продукты
- **🔄 Интерактивный чат** - Общение с агентом на естественном языке

## 📦 Установка

### 1. Клонирование репозитория

```bash
git clone [<repository-url>]
cd Sales-Forecasting-AI-Agent/AIAgent
```

### 2. Создание виртуального окружения

```bash
# Windows
python -m venv .venv
.venv\Scripts\activate

# Linux/macOS
python -m venv .venv
source .venv/bin/activate
```

### 3. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 4. Установка дополнительных зависимостей

```bash
pip install catboost
```

### 5. Настройка переменных окружения

Создайте файл `.env` в корневой директории проекта:

```bash
cp .env.example .env
```

Или настройте переменные окружения вручную (см. `.env.example`).

### 6. Быстрый запуск

Используйте скрипты для автоматического запуска:

```bash
# Linux/macOS
./start.sh

# Windows
start.bat
```

Скрипты автоматически:
- Активируют виртуальное окружение
- Проверяют наличие `.env` файла
- Запускают backend и frontend
- Проверяют работу компонентов

## 🔧 Настройка и конфигурация

### Добавление новых инструментов

1. Создайте новый файл в `backend/agent/tools/`
2. Используйте декоратор `@tool` из SmolAgents
3. Добавьте импорт в `backend/agent/tools/__init__.py`
4. Обновите промпт агента в `backend/agent/smol_agent.py`

### Добавление новых моделей

1. Создайте функцию в `backend/models/`
2. Добавьте импорт в `backend/models/__init__.py`
3. Обновите сервисы в `backend/services/`

## 📊 Модели прогнозирования

### Prophet (Facebook)
- Преимущества: Хорошо работает с сезонностью и трендами
- Использование: Автоматическое обнаружение сезонных паттернов

### SARIMA (Statsmodels)
- Преимущества: Классический статистический подход
- Использование: Стационарные временные ряды

### CatBoost (Yandex)
- Преимущества: Градиентный бустинг, устойчив к переобучению
- Использование: Сложные нелинейные зависимости

### Ансамбль
- Комбинирует все 3 модели для повышения точности
- Автоматический выбор лучшей модели
